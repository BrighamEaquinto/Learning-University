Theorem 2: Existence and Uniqueness Theorem
A linear system is consistent if and only if the rightmost column of the augmented matrix is not a pivot column-that is, if and only if an echelon form of the augmented matrix has no row of the form \(___stuff here___\). If a linear system is consistent, then the solution set contains either (i) a unique solution, when there are no free variables, or (ii) infinitely many solutions, when there is at least one free variable.

Vectors in \(\mathbb{R}^{2}\)

  • matrices with only one column is called a vector. Examples:
    • \(u = \left[ \matrix{3 \\ -1} \right]\)
    • \(v = \left[ \matrix{.2 \\ .3} \right]\)
    • \(w = \left[ \matrix{w_{2} \\ w_{3}} \right]\) where \(w_{2}\) and \(w_{3}\) are any real numbers
  • The set of al vectors wih two entries is denoted by \(\mathbb{R}^{2}\) (read as “r-two”).
  • Summing vectors is done horizontally (adding row-wise):

\[ \left[ \matrix{1 \\ 2} \right] + \left[ \matrix{2 \\ 5} \right] = \left[ \matrix{1 + 2 \\ -2 + 5} \right] = \left[ \matrix{3 \\ 3} \right] \]

Given a vector u and a real number c, the scalar multiple of u by c is the vector cu obtained by multiplying each entry in u by c. Example: \[ \text{if} u = \left[ \matrix{3 \\ -1} \right] \text{and } c = 5, \text{then } cu = 5\left[\matrix{3 \\ -1} \right] = \left[ \matrix{15 \\ -5 } \right] \]
