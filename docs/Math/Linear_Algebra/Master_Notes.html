Theorem 1: Uniqueness of the Reduced Echelon Form
Each matrix is row equivalent to one and only one reduced echelon matrix
Pivot Positions
in a matrix A is a location in A that corresponds to a leading 1 in the reduced echelon form of A.
Pivot Column
is a column of A that contains leading 1’s in the reduced echelon form
Theorem 2: Existence and Uniqueness Theorem
A linear system is consistent if and only if the rightmost column of the augmented matrix is not a pivot column-that is, if and only if an echelon form of the augmented matrix has no row of the form \(___stuff here___\). If a linear system is consistent, then the solution set contains either (i) a unique solution, when there are no free variables, or (ii) infinitely many solutions, when there is at least one free variable.
